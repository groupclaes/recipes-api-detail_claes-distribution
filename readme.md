# Claes Distribution Recipes Detail API Controller

## Endpoints
`GET /api/{version}/recipes/detail/:id`  
Query parameters {type} name [default value]:
- {string} culture ['nl']
- {number} shop_id [null]
`GET /api/{version}/recipes/detail/:id/qr-code.png`  
Query parameters {type} name [default value]:
- {string} culture ['nl']
